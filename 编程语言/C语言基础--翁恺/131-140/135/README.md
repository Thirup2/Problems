# 数字金字塔
输入正整数n，输出n层数字金字塔。

### 输入格式:
正整数n，1<=n<=9。

### 输出格式:
n层的金字塔，其中第1层的“1”在第1行的第39列（列从1开始计数）。

### 输入样例:
```
5
```
### 输出样例:
```
                                      1
                                     121
                                    12321
                                   1234321
                                  123454321
```
